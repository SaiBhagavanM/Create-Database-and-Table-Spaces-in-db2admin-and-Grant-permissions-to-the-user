Step 4: Create Database and Table Spaces and Grant permissions to the user 
1.	Open DB2 command window administrator (use Db2admin user) from start menu.
2.	Execute the below commands:
Db2
CREATE DATABASE DBSData AUTOMATIC STORAGE YES USING CODESET UTF-8 TERRITORY US 
UPDATE DB CFG FOR DBSData USING AUTO_MAINT ON
UPDATE DB CFG FOR DBSData USING AUTO_TBL_MAINT ON
UPDATE DB CFG FOR DBSData USING AUTO_RUNSTATS ON
update database config for DBSData using LOGFILSIZ 1024
update database config for DBSData using logsecond 40
update database config for DBSData using logprimary 40
update database config for DBSData using num_log_span 36
update database config for DBSData using stmt_conc LITERALS

CONNECT TO DBSData
CREATE SCHEMA DBSData AUTHORIZATION yourPassword

CREATE BUFFERPOOL YFSBUFFER1 IMMEDIATE  SIZE AUTOMATIC PAGESIZE 4 K 
CREATE  LARGE  TABLESPACE YFSTS1 PAGESIZE 4 K  MANAGED BY AUTOMATIC STORAGE  BUFFERPOOL  YFSBUFFER1 

CREATE BUFFERPOOL YFSBUFFER2 IMMEDIATE  SIZE AUTOMATIC PAGESIZE 8 K 
CREATE  LARGE  TABLESPACE YFSTS2 PAGESIZE 8 K  MANAGED BY AUTOMATIC STORAGE  BUFFERPOOL  YFSBUFFER2 

CREATE BUFFERPOOL YFSBUFFER3 IMMEDIATE  SIZE AUTOMATIC PAGESIZE 16 K 
CREATE  LARGE  TABLESPACE YFSTS3 PAGESIZE 16 K  MANAGED BY AUTOMATIC STORAGE  BUFFERPOOL  YFSBUFFER3 


CREATE BUFFERPOOL YFSBUFFER4 IMMEDIATE  SIZE AUTOMATIC PAGESIZE 32 K 
CREATE  LARGE  TABLESPACE YFSTS4 PAGESIZE 32 K  MANAGED BY AUTOMATIC STORAGE  BUFFERPOOL  YFSBUFFER4 

CREATE SYSTEM TEMPORARY TABLESPACE YFSTS1TMP PAGESIZE 4 K MANAGED BY AUTOMATIC STORAGE BUFFERPOOL YFSBUFFER1
CREATE SYSTEM TEMPORARY TABLESPACE YFSTS2TMP PAGESIZE 8 K MANAGED BY AUTOMATIC STORAGE BUFFERPOOL YFSBUFFER2
CREATE SYSTEM TEMPORARY TABLESPACE YFSTS3TMP PAGESIZE 16 K MANAGED BY AUTOMATIC STORAGE BUFFERPOOL YFSBUFFER3
CREATE SYSTEM TEMPORARY TABLESPACE YFSTS4TMP PAGESIZE 32 K MANAGED BY AUTOMATIC STORAGE BUFFERPOOL YFSBUFFER4
GRANT DBADM,CREATETAB,BINDADD,CONNECT,CREATE_NOT_FENCED_ROUTINE,IMPLICIT_SCHEMA,LOAD, CREATE_EXTERNAL_ROUTINE,QUIESCE_CONNECT,SECADM ON DATABASE TO USER MyUser. (MyUser is any system admin)

Relaunch the DB2 command window(use MyUser). (MyUser is same system admin for whom permissions were granted above)
DB2
CONNECT TO DBSData
GRANT  DBADM,CREATETAB,BINDADD,CONNECT,CREATE_NOT_FENCED_ROUTINE,IMPLICIT_SCHEMA,LOAD, CREATE_EXTERNAL_ROUTINE,QUIESCE_CONNECT,SECADM ON DATABASE TO USER DB2ADMIN
GRANT  CREATEIN,DROPIN,ALTERIN ON SCHEMA DBSData TO USER DB2ADMIN WITH GRANT OPTION
GRANT USE OF TABLESPACE YFSTS1 TO USER DB2ADMIN WITH GRANT OPTION
GRANT USE OF TABLESPACE YFSTS2 TO USER DB2ADMIN WITH GRANT OPTION
GRANT USE OF TABLESPACE YFSTS3 TO USER DB2ADMIN WITH GRANT OPTION
GRANT USE OF TABLESPACE YFSTS4 TO USER DB2ADMIN WITH GRANT OPTION
